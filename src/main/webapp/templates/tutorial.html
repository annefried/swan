Welcome to a quick tour of SWAN!


SWAN is designed such that most things should be self-explanatory. You can click around and try out things for yourself, or follow the tours below!

SWAN provides two different user roles. SWAN users are either project managers or annotators. Depending on whether you are logged in as a project manager or as an annotator, you will see the project manager or the annotator tours. If you are trying out SWAN for the first time, start with the project manager tour and then log in as an annotator (you will create an account during the tour).

HINT: In case a tour gets stuck, just go back to this tutorial page and refresh your browser. Then restart the tour. This is a known problem with the tutorial library we're using. We're aplogize and we are currently working to solve this problem. By the way... this doesn't happen if you carefully do what the tutorial tells you to do. :)



Before starting the tour, go to the Project Explorer and check whether some document is waiting for you. Otherwise, ask your project manager to add some documents for you.
